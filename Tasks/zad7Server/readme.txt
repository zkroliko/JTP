Targety anta: clean, build, run

Program reskaluje obrazki do odpowiedniego rozmaru podawanego w UI Clienta. Zapisywane są w odpowiednim katalogu przez serwer wraz z informacją odnośnie przebiegu. Komunikacja pomiedzy klientem a serwerem odbywa się za pomocą katalogu z obrazkami oraz pliku .ser. Ilosc watkow ustawia sie przy tworzeniu serwera.
